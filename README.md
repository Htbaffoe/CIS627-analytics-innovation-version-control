# Analytics Innovation & Version Control – Sports Decision Support Project

## Project Overview
This project proposes a sports analytics idea called the **Player Rotation Readiness Dashboard**. The purpose of the tool is to help coaching staff and front office decision makers evaluate whether a player is ready for increased minutes, should maintain their current role, or may need a reduced workload. The idea combines performance and workload indicators into one simple decision-support view.

## Decision-Making Problem
A major challenge in sports organizations is deciding how to manage player rotation and playing time throughout a season. These decisions affect team performance, player development, and possible fatigue or injury risk. The dashboard is intended to support more informed decisions about whether a player should receive more minutes, fewer minutes, or stay in the same role.

## Proposed Analytics Approach
The dashboard would use data such as minutes played, game frequency, shooting efficiency, scoring trends, and turnover rates. The analysis would focus on identifying patterns between workload and player performance over time. Rather than relying on one metric alone, the tool would combine several indicators to provide a balanced picture of player readiness.

## Use by Decision Makers
A coach could use this dashboard before games or during weekly planning to decide whether a player should receive increased, stable, or reduced playing time. A front office executive could use it to monitor player utilization across the season and support broader roster-management decisions. The goal is to make the information easy to interpret and directly useful in decision-making.
